## 标题
以`#`数量来区分标题的等级，`#`越多标题等级越大，字体也越小

## 换行
不是以换行符作为分割符，来实现换行功能的。这点与Word差别较大。
一个段落就是语义表达的一个单元，无需手动换行，MarkDown解释器
会更加显示框的大小自动排版。如果想另起一个段落，只需在结尾处连续输入两个空格或者回车符即可。

这样，解释器就知道，在这里需要另起一个段落。
否则它还是把上下文当做一行处理。

## 字体
文字两边用`**`包围起来，就是表述**加粗**；用`*`包围起来表示
*斜体*，用`~~`包围起来，就是表示~~删除~~。
